# Signal Habit Lab

Signal is a habit tracker prototype designed to compete against ordinary streak-first trackers.

The differentiator: it treats every check-in as evidence. Instead of only asking whether a habit was done, Signal captures the minimum viable habit, cue, friction, energy, place, and notes. It then turns that into best-condition insights and small experiments.

## Open The App

Open `index.html` in a browser. The app is static and stores demo data in the browser with local storage.

## What Makes It Different

- Anti-streak habit logging with `Full`, `Tiny`, and `Blocked` states.
- Friction, energy, and environment context for every check-in.
- Pattern finding that shows which conditions make habits easier.
- Smart experiments that suggest the next smallest design change.
- Exportable habit data for future analytics or account sync.
